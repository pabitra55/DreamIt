Angular Installation
  • Step 1
      Download Nodejs
      https://nodejs.org/en/download/
  • Step 2
      Check if Nodejs is Installed ?
      npm -v
  • Step 3
      Check if NPM is installed? (NPM:- Node Package Manager)
      npm -v
  • Step 4
      Install Angular CLI (Command Line Interface)
      npm install -g @angular/cli
      npm install -g @angular/cli@latest
      npm install -g @angular/cli@9 Specific to verson
  • Step 5
      Check if angular CLI is installed?
      ng v
  • Step 6
      Create a new angular project
      ng new <{{'project - name'}}>
  • Step 8
      Run the project
      Go to the project directory
      ng serve (project get up and run in server)
  • Step 9
      Open your browser (any) and open the given URL
      http://localhost:4200
      ng serve (project get up and run in server)
    • Note :   ng serve command launches the angular application, watches your file changes and rebuilds the app when you make changes to those files
    Bootstrap Installation
  • Step 1

      In the following steps we will add bootstrap 5 into our angular project

      There are numerous ways to install bootstrap in your angular project

      1. You can install Bootstrap using npm install command.
      2. Download Bootstrap file from its official site and keep them in the src/assets folder of angular project folder
      3. By using Bootstrap CDN

      Let's Start With NPM way Go to Command Line Interface and install latest Bootstrap through npm

      $ npm install Bootstrap

      The above command will add bootstrap package in package.json file. No need to worry all these jobs will be performed by angular CLI while installation

      The Bootstrap assets installed and added into node_modules/bootstrap folder

    Step 2

      Now we need to import css in to global style.css/style.scss

      Open the style.scss file from your project folder and try to import bootstrap.css file

      @import "~bootstrap/dist/css/bootstrap.css"
